HTML

HTML, Siglas en inglés de HyperText Markup Lenguage hace referencia al lenguaje de marcado para la elaboración de páginas web,no confundir con un lenguaje de programación.Es un estandar a cargo del

Word Wide Web Consortium (W3C)

organización dedicada a la estandarización de casi todas las tecnologías ligadas a la web.

Primeras especificaciones de HTML

Tim Bernes Lee en 19911 describe 18 elementos que incluyen el diseño inicial y relativamente simple HTML.

Bernes-Lee consideraba el HTLM una ampliación de SGML perono fue formalmente reconocida como tal hasta la publicación a mediados de 1993 por la IETF2 ,de una primera proposición para una específicación HTML: el borrador del Hypertex Markup Lenguaje de Bernes-Lee y Dan Connolly

Marcador HTLM

El HTML se escribe en forma de <etiquetas>, rodeadas por corchetes angulares (<>/) El HTML también puede describirse hasta cierto punto la apariencia de un documento y puede incluir o hacer referencia a un tipo de programa llamado script, el cual puede afecar el comportamiento de navegadores web.
